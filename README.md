# Homepage

A minimalistic personal link hub (Linktree alternative) with light/dark mode support.

## Features

- 🎨 Clean, minimalistic design
- 🌓 Light/dark mode toggle with automatic system preference detection
- 📱 Fully responsive layout
- ✨ Hover animations and staggered entrance effects
- ⚡ Pure HTML, CSS, and JavaScript (no frameworks)
- 🎯 Quick access to all social profiles

## Usage

Simply open `index.html` in your browser or deploy the root folder to any static hosting service.

### Customization

1. Edit `index.html` to update your name, bio, and links
2. Modify `style.css` to change colors and styling
3. Replace `favicon.ico` with your own favicon
